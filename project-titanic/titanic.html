Titanic Data Analysis

Project Brief:

This project aims to create a model to predict the probability of a given passengers surviving the titanic tragedy. The model uses information such as the passenger's age, sex, cabin number, etc.

A set of training data is provided. This data includes the whether or not each passenger survived, hence is used carry out the machine learning. Another set of data is used to test the accuracy of the model. The survival infomation is not visible to the project.

The analysis and model contruction is done using Python. The Pandas module is used extensively. This is a well known data science project. Data and details regarding this project can be found on Kaggle website.

Data Manipulation:

1. Dropping less useful feature

The Cabin Number and Ticket Number provides little correlation to the survival of the passengers. Therefore, they are dropped at this stage. However, they maybe added back in the future for further analysis.

2. Complete Missing Data

Due to the nature of the sklearn module, all data in every observation must be complete. Therefore it is better to complete the data before manipulating the data. This project fills in data by choosing the mode of the data in observations that have similar data in other features.

For example, the project estimates the age band of passengers based on other passengers with the same Pclass and gender.

3. Extract Title from passenger Name:

The Name feature in the training data contains the tile of the passenger. The number and the precentage survived of each title is shown in the table below. The survival % shows that there is strong correlation between the passenger's title and his/her survival probability.

6. Reformat data:

To accomodate for the sklearn module function format. All features in the test and trainign data are converted to ordinal values. All missing data is filled in using methods previously mentioned.

Create Model:

The Python sklearn module provides many predictive models in simple to use format. Little work is need in this section. Right now there are 2 models being used:

Logistic Regression

Logistic Regression model is applied to the data. The sklearn function is left at default values. The accuracy of this model is 80.36%. I have yet to fully understand the Logistic Regression model. When I do, I will tweak some values and try it again.

K-Nearest Neighbors

KNN model is applied to this data. The KNN model compares test observation data with other data with observations with similar dependent feature data, then predicts the survival value based on the survival data of those observations. The accuracy is at 83.95% when K value is set to 5.
